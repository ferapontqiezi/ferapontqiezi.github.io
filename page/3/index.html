python第三方库安装报错解决办法

发表于 2021-01-29 更新于 2021-02-02 分类于 python

两种可能原因

pip版本不够新

安装第三方库的时候pycharm报以下错误：

我们可以在pycharm中的文件-设置-解释器（Interpreter）中看看pip的版本是不是最新的，如果不是最新的可以进行更新（update）

如果像上图一样已经更新到最新版本，那我们可以用cmd窗口直接输入命令行来看看原因。

pip速度不够

输入命令行pip install qypt5，产生报错：1. socket.timeout: The read operation timed out

  1. pip._vendor.urllib3.exceptions.ReadTimeoutError: HTTPSConnectionPool(host=’files.pythonhosted.org’, port=443): Read timed out.

发生这两种异常（exceptions）的原因是读取时间超时，可见速度5.5kB/s，下载57MB的东西要2个多小时确实有问题。面对这种问题，我们可以换为更强大的pypi，但是由于国外官方pypi经常被墙，导致不可用，所以我们最好是将自己使用的pip源更换一下，这样就能解决被墙导致的装不上库的烦恼。常见的镜像网站：

  1. 清华大学：https://pypi.tuna.tsinghua.edu.cn/simple

  2. 阿里云：http://mirrors.aliyun.com/pypi/simple/

  3. 中国科技大学 https://pypi.mirrors.ustc.edu.cn/simple/

  4. 华中科技大学：http://pypi.hustunique.com

  5. 山东理工大学：http://pypi.sdutlinux.org/

  6. 豆瓣：http://pypi.douban.com/simple/
